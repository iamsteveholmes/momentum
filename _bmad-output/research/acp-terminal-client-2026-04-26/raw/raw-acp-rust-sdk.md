# Raw: ACP Libraries Page (agentclientprotocol.com/libraries/rust)

## Rust Library

The `agent-client-protocol` Rust crate provides implementations of both sides of the Agent Client Protocol that you can use to build your own agent server or client.

Install:
```toml
cargo add agent-client-protocol
```

You implement either the `Agent` trait or the `Client` trait to define the interaction with the ACP counterpart.

Example binaries:
- `agent.rs` — example agent server
- `client.rs` — example client

Documentation: https://docs.rs/agent-client-protocol/latest/agent_client_protocol/

### Users

The `agent-client-protocol` crate powers the integration with external agents in the Zed editor.
