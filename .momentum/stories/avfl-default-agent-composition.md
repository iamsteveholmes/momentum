## Story

As a AVFL orchestrator,
I want use a default agent composition for each AVFL profile (scan, checkpoint, deep) without specifying team members each time,
so that AVFL invocations require no per-invocation team composition decisions — profiles carry sensible defaults.

## Description

Every AVFL invocation requires specifying which agents to include. Default compositions per profile (scan: enumerator only; checkpoint: enumerator + adversary; deep: enumerator + adversary + fixer + domain expert) would reduce orchestration overhead and prevent composition errors.

**Pain context:** Nornspun upstream #11 (Low). Composition errors happen frequently. Default compositions reduce the decision surface to profile selection only.
