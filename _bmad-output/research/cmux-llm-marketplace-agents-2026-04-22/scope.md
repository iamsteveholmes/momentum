# Research Scope: CMUX-integrated coding agents with LLM marketplace support

**Date:** 2026-04-22
**Profile:** light
**Goals:** Identify the best CLI coding agents that (a) have first-class CMUX integration, (b) access LLM marketplaces like OpenRouter and OpenAI-compatible providers, and (c) could realistically replace Claude Code — with comparable support for sub-agents, slash commands/skills, hooks, MCP servers, tool use, and plan mode. Excludes Goose and ForgeCode. Obvious candidates to evaluate: opencode, pi.

## Sub-Questions

1. Beyond opencode, pi, Goose, and ForgeCode, what other CLI coding agents exist that could serve as Claude Code replacements, and how deeply does each integrate with CMUX (native pane control, shell-wrapper, or none)?
2. How do the top candidates support LLM marketplaces like OpenRouter, OpenAI-compatible endpoints, and arbitrary custom providers? What is the provider and model flexibility, and can routing be configured per-task?
3. How do these agents compare to Claude Code on feature parity across: sub-agents, slash commands/skills, hooks, MCP servers, tool use, plan mode, and overall ecosystem maturity?
