# Eval: Quick-Fix Gate Failure Warnings Render in Blockquote

Given a quick-fix workflow that encounters a gate failure (e.g., AVFL GATE_FAILED or missing guidelines for specialist), the output should render the warning in a `>` blockquote so it stands out visually from surrounding text.
